# CSIS3126_Wayoi
Design Project I

Problem Statement:
Current quiz and educational game platforms (like Kahoot) require paid subscriptions for full features, have limited customization options, and often lack flexibility for different use cases. 
This project aims to create an open, customizable quiz platform that educators, trainers, and hobbyists can use for free to create engaging, interactive quizzes for classrooms, training sessions, 
trivia nights, or study groups.

Technology Stack:
Frontend: HTML, CSS, JavaScript (with Bootstrap for styling);
Backend: Python with Flask framework;
Database: MySQL;
Real-time Communication: Flask-SocketIO for live game sessions;
Authentication: Flask-Login with Werkzeug for password hashing.

Platform Focus:
Web-based platform accessible on both desktop and mobile browsers (responsive design), rather than building separate native mobile apps.

Scope:
Core quiz creation and management;
Live game hosting and gameplay;
User authentication (host/player roles);
Real-time leaderboards;
Basic statistics and game history;
In-app documentation.

The plan is to build the core functionality first (quiz creation, gameplay with page refreshes) and add the real-time features with Flask-SocketIO once the foundation is solid.
