Platform Help & Documentation

How to Create a Quiz

1. Register or log in to your account.

2. From the Dashboard, click Create New Quiz.

3. Add a Title and Description, then click Create.

4. Add questions (Multiple Choice, True/False, or Fill in the Blank).

How to Host a Game

1. On your Dashboard, find the quiz you want to host.

2. Click the Host Live Game button.

3. A unique 6-character Session Code will be generated.

4. Ask players to join using this code! Wait for them to appear in the lobby, then click Start Game.

How to Join a Game

1. Go to the homepage (no account required).

2. Enter the Session Code provided by the host.

3. Choose a Nickname and click Join Game.

4. Wait in the lobby until the host starts the game!
